Набор инструментов для работы с веб-камерой: запись видео, таймлапс, настройка временных меток и сборка кадров в видео.

## ✨ Возможности

| Файл | Описание |
|------|----------|
| `webcam_recorder.py` | Простая запись видео с веб-камеры. Управление клавишами R (запись) и Q (выход). |
| `webcam_recorder_pro.py` | Продвинутая версия с графическим интерфейсом. Запись видео, скриншоты, таймлапс, пауза. |
| `timestamp_settings.py` | Настройка отображения времени на кадре: шрифт, цвет, позиция, тень. |
| `simple_timelapse.py` | Таймлапс с настраиваемым интервалом. Сохраняет кадры в папку `timelapse/`. |
| `timelapse_to_video.py` | Сборка кадров таймлапса в видео (MP4). |
Webcam Recorder Suite/

├── timelapse_to_video.py       # Сборка видео из кадров
├── videos/                     # Папка для записанных видео
├── screenshots/                # Папка для скриншотов
├── timelapse/                  # Папка для кадров таймлапса
├── timelapse_videos/           # Папка для готовых видео
└── README.md
Управление
Клавиша / Кнопка	Действие
R	Начать/остановить запись
Q / ESC	Выход
GUI кнопки	Запись, скриншот, таймлапс
